# HTML CV Project

A simple project for creating a **Curriculum Vitae (CV)** using only **HTML**.  
This project allows you to generate a clean, professional CV webpage that can be viewed in a browser or exported as a PDF.

---

## Overview

This project uses a single HTML file to structure and display CV information such as personal details, education, experience, and skills.  
The goal is to keep the project **simple, portable, and easy to customize** without relying on frameworks or complex tools.

---

## Features

- Pure **HTML implementation**
- Easy to edit and customize
- Lightweight and dependency-free
- Print-friendly
- Can be exported as **PDF from the browser**
- Suitable for **personal websites or GitHub Pages**
